# Project 2 - PHI Redaction Pipeline

This project automatically detects and hides sensitive 
patient information from clinical notes.

## What it does
- Hides phone numbers
- Hides emails
- Hides SSN
- Hides dates
- Hides patient names
- Hides doctor names
- Hides medical record numbers
- Hides addresses and zipcodes
